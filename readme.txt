Hii, there...

* To start the app simply write the following command on terminal :)

1)npm install

2)npm start

your app will start at port no. 3003 

-----------------------------------------------------------------------

* You can create users or use the dummy users 

1)To see the dummy users simply do the `get` request at url `127.0.0.1:3003/api/user`

2)To create user simply do a `post` request at url `127.0.0.1:3003/api/user`

------------------------------------------------------------------------

* Test the app

1)Simply do a `patch` request at url `127.0.0.1:3003/api/user/userId`

Note** You will get the response in JSON format

2)After doing `patch` request, you can do the get request to the updated data
